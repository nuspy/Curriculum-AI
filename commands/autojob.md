description: Panoramica AutoJob e verifica che il sistema sia pronto all'uso.
---
Usa la skill `autojob`. Per prima cosa assicurati che il daemon sia attivo: se i tool MCP del server
`autojob` non rispondono, esegui in shell `uv run autojob up` nella cartella del progetto AutoJob.
Poi riepiloga brevemente all'utente cosa puoi fare (ricerca multi-portale, ranking sul CV, cover
letter/risposte, compilazione candidature in modalità manuale) e chiedi cosa desidera. Se utile,
indica il cockpit su http://127.0.0.1:8765/.
